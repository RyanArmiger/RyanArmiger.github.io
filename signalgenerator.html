MBED SIGNAL GENERATOR

Technology: C++, MBed, CircuitMaker, ARM MBed Compiler, OLED Display, Capacitive Switches 

TECHNOLOGY

Features

Simple low-to-medium frequency signal generator for square, sawtooth and sine waves. Features capacitive switches and an OLED display with a GUI programmed on an Mbed. 

OLED display and GUI

Black and white OLED display with convenient GUI, displays frequency of signal being generated and allows selection between square, sawtooth and sine waves using images. Also contains a game where the user can draw images using the switches like an etch-a-sketch

Signal generator

Capacitive switches allow the user to interact with the GUI and change settings such as signal frequency. Interrupt handling reduces the effect running the GUI has on the signal generated, negligible at lower frequencies. 

Technology

CircuitMaker files for schematics and PCB available here. Mbed code here

  • Hardware
  • Software

Schematics were created following lab instructions and the PCB was designed in CircuitMaker. PCB contains pads which, when pressed, change capacitance. By choosing appropriate resistors and capacitors within the circuit, these act as switches for the Mbed microcontroller.

The Mbed microcontroller is programmed in C++ and compile into bin files using the online ARM Mbed compiler. This software handles the switch interrupts, allowing the GUI to be displayed and used while a signal is being generated simultaneously.
