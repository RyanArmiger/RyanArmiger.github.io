Innovative app for diabetes management

 Technology: Swift, Xcode, MacOS, Jazzy, GitHub

TECHNOLOGY DOCUMENTATION

Features

Created for the Institute of Biomedical Engineering at Imperial College London over 6 weeks, the app provides a simple yet detailed interface to allow diabetics to improve management of their condition.  

Bifocal Display

The user’s blood glucose level and recommended insulin doses are displayed on a bifocal display. Low/avg/high glucose values for previous days are displayed on sidebands.

Habit Logging

The food and exercise sections allow the user to input information about meals and exercise activities respectively, and includes a favouriting system for convenience.

Intuitive UI

The lack of navigational hierarchy allows all information to be accessed with one click and always-visible status indicators ensure all important information is visible.

Summary Info

Each day's meal and exercise logs are displayed and the health section lets the user view and filter through summaries of previous day's logs. 

Design Process

Communication with the client

Weekly meetings ensured the iterative process of development remained true to the aims of the client, while allowing the development team to offer practical, aesthetic and technically viable solutions.

Focus groups with Diabetics

Multiple focus groups with diabetics allowed critical input from target users on the usefulness and priority of the app's features.

User Interface

The navigation between domains within the app is handled by hiding and displaying embedded views. This allows each view to be controlled by a separate view controller, greatly simplifying development.
Communication between view controllers is handled via iOS' radio system, the Notification Centre, which alerts view controllers when their view needs to be updated and passes small amounts of information.

Relational Database

A relational database is used to connect and store the user's habit logs. These are persisted using iOS device's "Core Data". The relationships between objects and extension methods created allow simple, intuitive access to user information.
