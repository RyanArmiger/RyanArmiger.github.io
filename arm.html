ARM ASSEMBLY SIMULATOR

Technology: F#, HTML, CSS, Javascript, Fable, Electron, Visual Studio Code, VisUAL, GitHub

TECHNOLOGY

Features

Simulates running of ARM Assembly instructions with a convenient GUI to display memory location contents and flags set, created as part of a Third-Year programming module.

Code Simulation

Well-tested and almost perfect simulation of ARM Assembly instructions, including branch and EQU instructions. 

Useful GUI

Convenient displaying of memory locations in decimal, hexadecimal and binary. Also displays flags set, and instruction cycle.

File Management

Saving and loading of .txt files containing lines of ARM instructions for convenient development and education. 

Technology

GitHub project available on request

  • GUI
  • MODEL

Written in F# and compiled using Fable to write Javascript code. Electron then used to create a cross-platform GUI. Monaco editor is incorporated to allow ARM Assembly lines to be input and provides syntax highlighting. Other elements are written in HTML, CSS and F#. 

The model for simulating ARM instructions is written in F#, using functional programming to parse instruction lines and execute them on simulated memory locations, with minimal boilerplate code. Testing done in F# using FsCheck and unit tests against VisUAL, an existing simulator written in Java.
